WORLD 设计文档

以下网页记录 WORLD 已确认的产品设计方向。结构和边界可以作为开发输入，但未明确锁定的数值、概率、资产、Milestone 或发布计划仍需独立审核。

  • 模式总览

    模式定位、长期状态、行动、区域、成长、经济与仓库边界。

  • 怪物卡属性、成长资质与交易

    四项显式战斗属性、隐藏成长资质、无职业技能成长、插画基因、实体卡快照和数字所有权交易。

  • 事件、战斗随机与怪物迭代

    属性事件检定、有限战况随机、5对8等多人结算、前线宽度、繁殖边界与非吞噬式传承。

  • 异步领地冲突

    临时占领、时间线汇合、一次性抢地和历史收益不回滚。

  • 开发难度与路线

    系统难度、Simulation 风险和推荐开发顺序。

  • 怪物角色与驻军来源

    账号内怪物、养成与招募两种驻军来源、地点容量和角色保护。

  • MVP 内容规模与 Event Composer

    事件模板、视觉资产、数据对象、MVP 规模和内容生产工具。

  • 持续车怪物卡

    每次选择持续塑造属性、技能、特性、经历、关系与世界记录。
