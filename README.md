# Realm Ruckus WORLD

`realmruckus/world` is the standalone WORLD MVP game and product website repository.

The repository root is the website root:

- `index.html` — product home
- `play.html` — playable vertical slice
- `architecture.html` — repository and runtime architecture
- `docs/` — all WORLD design documents as web pages
- `releases.html` — version history
- `status.html` — long-lived product status
- `js/engine.js` — browser JavaScript game-engine prototype

WORLD consumes approved shared inputs from `realmruckus/core` through `dependencies.lock.json`. Dynamic tasks, priorities, blockers and milestones are maintained in Linear.
